# Objetivo del producto

El objetivo del Sistema de Reservas de Zonas Comunes es proporcionar una plataforma digital que permita gestionar de forma organizada, transparente y eficiente las reservas de los espacios comunes del Conjunto Residencial Peñas Blancas.

La solución deberá permitir:

* Consultar disponibilidad en tiempo real.
* Solicitar reservas de manera sencilla.
* Aprobar o rechazar solicitudes.
* Administrar zonas comunes y horarios.
* Llevar un historial de reservas.
* Reducir la carga operativa de la administración.
* Mejorar la experiencia de los residentes.
* Sentar las bases para futuras funcionalidades y automatizaciones.
