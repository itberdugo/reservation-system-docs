# Problema actual

Actualmente, para realizar la reserva de una zona común, como el auditorio, salón social o cualquier otro espacio disponible, el residente debe comunicarse directamente con la administración del conjunto.

La administradora debe verificar manualmente la disponibilidad, registrar la solicitud y comunicar la respuesta al residente.

Este proceso presenta varios inconvenientes:

* Dependencia de una sola persona.
* Falta de disponibilidad en tiempo real.
* Posibles cruces o dobles reservas.
* Poca trazabilidad de las solicitudes.
* Procesos manuales que consumen tiempo.
* Dificultad para consultar el historial de reservas.
* Comunicación poco eficiente entre residentes y administración.

Debido a estas limitaciones surge la necesidad de implementar una solución tecnológica que permita automatizar y centralizar el proceso de reservas.
