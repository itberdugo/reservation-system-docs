# Diagramas

Los diagramas permiten representar gráficamente los procesos y componentes principales del sistema.

## Diagramas contemplados

- Diagrama de actores.
- Diagrama de casos de uso.
- Diagrama de flujo de reserva.
- Diagrama de flujo de aprobación.
- Diagrama de secuencia.
- Diagrama de clases.
- Modelo entidad-relación.

Estos diagramas serán desarrollados durante las siguientes etapas del proyecto.
