# Historias de Usuario

## HU-001 Consultar disponibilidad

Como residente, quiero consultar la disponibilidad de las zonas comunes para conocer las fechas y horarios libres antes de realizar una reserva.

---

## HU-002 Solicitar reserva

Como residente, quiero solicitar una reserva para utilizar una zona común en una fecha específica.

---

## HU-003 Aprobar solicitudes

Como administrador, quiero aprobar o rechazar solicitudes para controlar el uso de las zonas comunes.

---

## HU-004 Configurar horarios

Como administrador, quiero configurar horarios disponibles para organizar el uso de cada espacio.

---

## HU-005 Ver reservas aprobadas

Como personal de vigilancia, quiero consultar las reservas aprobadas para controlar el acceso de los residentes.

---

## HU-006 Consultar reportes

Como miembro del consejo administrativo, quiero consultar reportes para realizar seguimiento al uso de las zonas comunes.
