# Actores

Los actores representan los usuarios o entidades que interactúan con el sistema.

## Residente

Usuario encargado de consultar disponibilidad y solicitar reservas de zonas comunes.

### Responsabilidades

* Iniciar sesión.
* Consultar disponibilidad.
* Solicitar reservas.
* Cancelar reservas.
* Consultar historial.

---

## Administrador

Usuario encargado de gestionar el funcionamiento del sistema.

### Responsabilidades

* Aprobar reservas.
* Rechazar solicitudes.
* Gestionar zonas comunes.
* Configurar horarios.
* Consultar reportes.

---

## Vigilancia

Usuario encargado del control de acceso.

### Responsabilidades

* Consultar reservas aprobadas.
* Verificar acceso a las zonas comunes.

---

## Consejo Administrativo

Actor con acceso a información y reportes.

### Responsabilidades

* Consultar estadísticas.
* Revisar reportes.
* Supervisar el uso de las zonas comunes.
