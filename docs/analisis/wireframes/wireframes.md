# Wireframes

Los wireframes representan una aproximación visual de las pantallas del sistema.

## Interfaces consideradas

- Inicio de sesión.
- Dashboard principal.
- Calendario de reservas.
- Solicitud de reserva.
- Historial de reservas.
- Panel administrativo.
- Gestión de zonas comunes.
- Configuración de horarios.

Los diseños detallados serán desarrollados durante la fase de diseño.
