# Análisis

## Definición

La fase de análisis tiene como objetivo transformar los requerimientos del negocio en una representación funcional de la solución.

Durante esta etapa se identifican los actores involucrados, los casos de uso, las historias de usuario y los principales elementos del dominio del sistema.

La información obtenida servirá como base para el diseño de la arquitectura y la posterior construcción del producto.

---

## Objetivos

* Identificar los actores del sistema.
* Definir los procesos principales.
* Describir las funcionalidades desde la perspectiva del usuario.
* Comprender el dominio del negocio.
* Establecer una base para el diseño y desarrollo.

---

## Componentes del análisis

* Actores.
* Casos de uso.
* Historias de usuario.
* Diagramas.
* Wireframes.
* Modelo del dominio.
