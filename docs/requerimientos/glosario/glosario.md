# Glosario

## Residente

Persona que vive dentro del conjunto residencial.

## Zona común

Espacio compartido disponible para reserva.

## Reserva

Solicitud realizada por un residente para utilizar una zona común.

## Administración

Personal encargado de gestionar las reservas.

## Vigilancia

Personal encargado del control de acceso.

## Estado de reserva

Situación actual de una reserva dentro del sistema.
