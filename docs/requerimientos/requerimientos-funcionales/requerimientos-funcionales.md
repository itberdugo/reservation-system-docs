# Requerimientos funcionales

## RF-001 Registro de usuarios

El sistema deberá permitir registrar residentes.

## RF-002 Inicio de sesión

El sistema deberá permitir autenticar usuarios.

## RF-003 Consulta de disponibilidad

El sistema deberá mostrar la disponibilidad de las zonas comunes.

## RF-004 Solicitud de reserva

El residente podrá solicitar una reserva.

## RF-005 Aprobación de reservas

La administración podrá aprobar o rechazar solicitudes.
