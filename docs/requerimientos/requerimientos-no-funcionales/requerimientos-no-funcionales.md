# Requerimientos no funcionales

## RNF-001 Disponibilidad

El sistema deberá estar disponible las 24 horas del día.

## RNF-002 Seguridad

Las contraseñas deberán almacenarse cifradas.

## RNF-003 Rendimiento

Las consultas deberán responder en menos de 5 segundos.

## RNF-004 Auditoría

El sistema deberá conservar registros históricos de las operaciones.

## RNF-005 Escalabilidad

La solución deberá permitir incorporar nuevas funcionalidades sin afectar las existentes.
